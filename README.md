# 📈 Machine Learning Trading Bot (Bybit)

[![Python 3.10+](https://img.shields.io/badge/python-3.10+-blue.svg)](https://www.python.org/downloads/)
[![CatBoost](https://img.shields.io/badge/CatBoost-Optimized-yellow.svg)](https://catboost.ai/)
[![Bybit API](https://img.shields.io/badge/Exchange-Bybit-black.svg)](https://bybit.com)

Автоматизированная система алгоритмического трейдинга для криптовалютной биржи Bybit. Проект реализует полный цикл количественных исследований (Quant Research): от разведочного анализа данных (EDA) до деплоя ML-моделей в виде REST API для генерации торговых сигналов в реальном времени.

Особенностью проекта является переход от классических метрик машинного обучения (Accuracy/F1) к **прямой оптимизации финансового результата (PnL)** с учетом торговых комиссий, проскальзывания и уникального альфа-фактора `rd_value`.

## 🗂 Структура репозитория

Проект разделен на исследовательскую среду (Jupyter Notebooks) и продакшен-сервер (Endpoints).

### 🔬 Research (Исследования и ML)
Ноутбуки демонстрируют эволюцию разработки торговой стратегии:

* **`01_bybit_EDA.ipynb`** — Разведочный анализ данных (Exploratory Data Analysis). Парсинг исторических OHLCV-данных с Bybit, очистка от аномалий, генерация лагов и инжиниринг признаков (включая обработку проприетарного сигнала `rd_value`).
* **`02_bybit_classification.ipynb`** — Базовые ML-модели. Обучение алгоритмов (CatBoost, Deep MLP, LSTM, Transformer) на задачу бинарной классификации (угадывание направления цены). Демонстрация проблемы «иллюзии высокой точности» (overtrading) на минутных таймфреймах.
* **`03_bybit_pnl.ipynb`** — Оптимизация на реальную прибыль. Внедрение финансового симулятора. Переход к PnL-ориентированной оценке моделей с использованием байесовской оптимизации (Optuna) для подбора идеального порога входа (Return Threshold), отсекающего рыночный шум.

### ⚙️ Production (API Эндпоинты)
Микросервисы для приема рыночных данных и возврата торговых решений:

* **`endpoint/`** — Актуальная реализация торгового API (v2). Содержит пайплайн загрузки лучшей обученной модели (CatBoost), расчета скользящих окон (Sliding Windows) в реальном времени и выдачи сигналов (Long/Short/Hold) с учетом комиссий taker/maker.
* **`old_endpoint/`** — Устаревшая версия API (v1) и ее документация. Оставлена для обратной совместимости или A/B тестирования старых стратегий.

## 🚀 Быстрый старт

### Требования
* Python 3.10+
* Установленные зависимости (CatBoost, Pandas, FastAPI/Flask, Optuna)

### Установка

1. Склонируйте репозиторий:
```bash
git clone git@github.com:vldzh/trading_bot.git
cd trading_bot
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Изучите исследования:
Рекомендуется начать с директории ноутбуков (`01` -> `02` -> `03`), чтобы понять логику генерации признаков и почему градиентный бустинг превзошел полносвязные нейросети на табличных данных.

4. Запуск торгового сервера:
Перейдите в папку актуального эндпоинта и запустите сервер:
```bash
cd endpoint
python main.py
```
